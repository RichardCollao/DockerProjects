

# COMANDOS UTILES

# Levantar servicios:
```html
docker compose up -d --build
```
```html
docker compose up -d --force-recreate
```
# Ver logs:
```html
docker compose logs -f app
```
```html
docker compose logs -f sonarqube
```
# Entrar al contenedor:
```html
docker exec -it sonar_app bash
```
